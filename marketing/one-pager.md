# Vaultline — Enterprise One-Pager

**AI Training Media Governance**

## The problem (right now)

Voice and video AI teams are scaling training data faster than they can govern it. Legal, procurement, and regulators are asking questions teams cannot answer from a shared drive:

- Which file is canonical?
- What transforms happened before training?
- Do we have license and consent on every clip?
- What exact dataset version shipped to production?

Spreadsheets and Slack threads are not surviving enterprise audits or EU AI Act documentation requirements.

## What Vaultline is

Vaultline is the **compliance and release control plane** for voice and video training media.

Creative teams keep working in Shotcut, Lightworks, Ardour, and LMMS. Vaultline catalogs every asset, runs automated QC gates, enforces compliance metadata, and ships **immutable, versioned release bundles** with full provenance.

## Core capabilities

| Capability | Outcome |
|------------|---------|
| Asset catalog + lineage | Answer "where did this clip come from?" in seconds |
| Policy-driven QC | Block bad audio/video before it hits training |
| Compliance metadata | License, consent, speaker ID enforced per profile |
| Immutable releases | Cite exact splits in papers, contracts, audits |
| Enterprise API | Integrate with existing MLOps and data stacks |
| Multi-format export | JSONL, HuggingFace, WebDataset |
